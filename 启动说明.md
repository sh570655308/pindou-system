# 启动说明

## 快速开始

### 1. 安装依赖

在项目根目录运行：

```bash
npm run install-all
```

这将安装后端和前端的所有依赖。

### 2. 启动开发服务器

```bash
npm run dev
```

这将同时启动：
- 后端服务器：http://localhost:5000
- 前端开发服务器：http://localhost:3000

### 3. 访问应用

打开浏览器访问：http://localhost:3000

## 默认账号

- **管理员账号**: `admin` / `admin123`
- 普通用户可以通过注册页面创建新账号

## 功能说明

### 主面板模块
系统采用模块化设计，登录后进入主面板，可以选择以下模块：

1. **库存管理模块** (`/inventory`)
   - 查看库存统计（总数量、种类数、低库存预警）
   - 按类别筛选产品（MARD、COCO等）
   - 搜索产品（按色号/代码）
   - "只看有货"筛选开关
   - 增加/减少库存数量（批量操作）

2. **图纸档案管理模块** (`/drawings`)
   - 管理拼豆图纸档案
   - 查看和编辑图纸信息
   - （功能开发中）

3. **拼豆消耗统计工具模块** (`/consumption`)
   - 统计拼豆消耗情况
   - 生成报表和分析
   - （功能开发中）

### 管理员功能
- ✅ 所有普通用户功能
- ✅ 系统管理面板 (`/admin`)
  - 管理库存大类（添加、编辑、删除）
  - 管理产品细类（添加、编辑、删除、设置颜色）
  - 设置预警阈值

## 技术栈

- **后端**: Node.js + Express + SQLite
- **前端**: React + TypeScript + Tailwind CSS
- **认证**: JWT Token

## 注意事项

- 首次运行会自动创建 SQLite 数据库（database.sqlite）
- 数据库会自动初始化默认管理员账号和示例数据
- 生产环境部署时请修改 JWT_SECRET 密钥
